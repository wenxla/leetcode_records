# 9. Palindrome Number

> [LeetCode Link](https://leetcode.com/problems/palindrome-number/) | Difficulty: **Easy** | Tags: Math

## Problem
Given an integer `x`, return `true` _if_ `x` _is a_ _**palindrome**__, and_ `false` _otherwise_.

palindrome: 回文 
example: 12321 ✅

## Thinking Process
1. 转成list左右指他
2. 转成反转的num
3. （🌟）把x对半分，通过比较rev和num大小，判断循环结束。 

## Solution

### Approach:
**Time:** O(1)
**Space:** O(logx)

```java
public boolean isPalindrome(int x) {
        if (x<0 || (x!=0 && x%10==0)) return false;
        int rev = 0;
        while (x>rev){
            rev = rev*10 + x%10;
            x = x/10;
        }
        return (x==rev || x==rev/10);
    }
```
